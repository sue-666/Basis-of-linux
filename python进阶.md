    
   
###### 转义字符:    
   
\'  表示一个单引号  
   
\"  表示一个双引号  
   
\n  表示一个换行符  
   
\\  表示一个反斜线   
   
\t  表示一个缩进, 相当于在键盘上按下了一个 tab 键  

####### 字符串拼接:  
name + "的年龄是" + age ="小明的年龄是18"   


###### 运算： 

 +       加              2 + 1 = 3  
 -       减              2 - 1 = 1    
 *       乘              1 * 2 = 2   
 /       除              1 / 2 = 0.5  
 %       取余            3 % 2 = 1  
 //      取整            7 // 2 = 3  
 **      乘方            3 ** 2 = 9
